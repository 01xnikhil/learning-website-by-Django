# Language Learning Website (Django)

## Overview
This is a **Language Learning Website** built using **Django**.  
It demonstrates Django skills by implementing features such as:

- Course modules (lessons, quizzes, exercises)
- User authentication (signup/login/logout/Forgetting Password)
- Progress tracking for learners
- Interactive exercises
- Responsive and clean UI

This project showcases Django best practices including models, views, templates, forms, and user session management.

---

## Features

1. **User Authentication**
   - Secure signup and login
   - Password hashing and validation

2. **Course Modules**
   - Organized lessons and quizzes
   - Progress tracking for each user

3. **Interactive Exercises**
   - Quizzes with instant feedback
   - Practice exercises per lesson

4. **Admin Panel**
   - Add/edit/delete courses and lessons
   - Monitor user progress

---

## Installation

### Requirements

- Python 3.x
- Django 4.x
- Other dependencies in `requirements.txt`

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/01xnikhil/learning-website-by-Django.git
   cd learning-website-by-Django

### Create a virtual environment:

python -m venv venv

Mac/ios -
source venv/bin/activate

windows -
venv/bin/activate

### Install dependencies:

pip install -r requirements.txt

### Apply migrations:

python manage.py migrate

### Run the server:

python manage.py runserver

### Open in browser:

http://127.0.0.1:8000/
