# 🚀 Language Learning Website (Django)

## 📌 Overview

The **Language Learning Website** is a web-based application built using **Django**, designed to help users learn new languages through structured courses, interactive exercises, and progress tracking.

This project demonstrates strong understanding of Django fundamentals including models, views, templates, forms, and user authentication. It provides a clean and user-friendly interface to enhance the overall learning experience.

---

## 🎯 Features

### 🔐 User Authentication

* Secure signup, login, and logout functionality
* Password hashing and validation
* Forgot password feature

### 📚 Course Modules

* Well-structured lessons and quizzes
* Organized course content
* Easy navigation between lessons

### 🧠 Interactive Exercises

* Practice exercises for each lesson
* Quiz system with instant feedback
* Helps reinforce learning

### 📊 Progress Tracking

* Tracks user progress in courses
* Displays learning status
* Personalized learning experience

### 🛠️ Admin Panel

* Add, update, and delete courses
* Manage lessons and quizzes
* Monitor user activity and progress

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **Authentication:** Django Auth System

## ⚙️ Installation & Setup

### 📋 Requirements

* Python 3.x
* Django 4.x
* Other dependencies in `requirements.txt`

---

### 🚀 Steps to Run

1️⃣ Clone the repository

```bash
git clone https://github.com/01xnikhil/learning-website-by-Django.git
cd learning-website-by-Django
```

2️⃣ Create virtual environment

```bash
python -m venv venv
```

👉 Activate environment

**Mac/Linux:**

```bash
source venv/bin/activate
```

**Windows:**

```bash
venv\Scripts\activate
```

3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

4️⃣ Apply migrations

```bash
python manage.py migrate
```

5️⃣ Run server

```bash
python manage.py runserver
```

6️⃣ Open in browser

```
http://127.0.0.1:8000/
```
